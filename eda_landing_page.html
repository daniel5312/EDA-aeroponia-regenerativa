Agricultura Regenerativa y Sostenible

EDA representa la evolución hacia un sistema agrícola inteligente que combina la eficiencia de la aeroponía con sensores IoT para crear un ecosistema de cultivo completamente automatizado y sustentable.

Sistema Aeropónico

Cultivo sin suelo utilizando nebulización de nutrientes directamente en las raíces, maximizando la absorción y minimizando el desperdicio.

Sensores IoT

Monitoreo continuo de humedad, pH, temperatura y nivel de nutrientes con 8 sensores especializados para optimización automática.

Sustentabilidad

Sistema de recirculación de agua y nutrientes, cero emisiones tóxicas, y estructura completamente reciclable.

Componentes Tecnológicos

Estructura Metálica Especializada

Fabricación con tubos estructurales HR50 y perfiles tubulares de diferentes diámetros, diseñada para soportar el sistema de cultivo vertical y garantizar estabilidad.

Sistema de Pulverización

8 pulverizadores de agua de 1/2" con bomba de 1200 Lt/h para distribución uniforme de nutrientes mediante nebulización controlada.

Sensores de Monitoreo

8 sensores de humedad relativa HS1101 y sensor de nivel de líquido para control automático de las condiciones de crecimiento.
